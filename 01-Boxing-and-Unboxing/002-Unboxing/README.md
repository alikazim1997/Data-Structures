# 02 - Unboxing

## 📌 Description

This project demonstrates the **Unboxing** concept in C#.

Unboxing is the process of converting an `object` back to its original value type.

## Example

```csharp
object obj = 10;
int number = (int)obj; // Unboxing
```

## Concepts Covered

- Boxing
- Unboxing
- Explicit Casting
- Value Types

## Technologies

- C#
- .NET Console Application
